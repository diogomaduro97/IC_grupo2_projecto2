# IC_grupo2


## How to run

* Audio_work:
  -Instalation of the Adam Stark's library:$ cd Audio_work
  $ git clone git@github.com:adamstark/AudioFile.git
  $ cd AudioFile
  $ mkdir build && cd build
  $ cmake ../
  $ make

  -To run Audio_work:
  
  $ cd Audio_work
  
  $ mkdir build && cd build
  
  $ cmake ../
  
  $ make
  
  $ ./audioTest [arg1][arg2]  
* opencv_work: 
  
  $ cd opencv_work
  
  $ mkdir build && cd build
  
  $ cmake ../
  
  $ make
  
  $ ./opencvTest [arg1][arg2]
